Photo Patchwork Printing

Description

Photo Patchwork Printer is a photo printing software.
It is able to print a set of photos on a page.

Features

The following features are planned for the first release. Since current version is alpha, only bold features are supported yet.

  • Resize, crop, rotate and flip photos
